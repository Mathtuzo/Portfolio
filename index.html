MES COMPÉTENCES

___________

HTML (HyperText Markup Language) est le langage de balisage standard utilisé pour créer et structurer le contenu d'une page web. Il permet de définir la structure logique et les éléments visuels d'une page en utilisant des balises et des attributs. HTML est interprété par les navigateurs web pour afficher le contenu d'une page de manière cohérente et compréhensible pour les utilisateurs.

HTML

Confirmé

CSS (Cascading Style Sheets) est un langage de feuilles de style utilisé pour contrôler l'apparence visuelle et la mise en page des éléments HTML sur une page web. Il permet de définir des règles de style telles que les couleurs, les polices, les marges, les espacements et les animations pour personnaliser l'aspect et le comportement des éléments HTML.

CSS

Confirmé

JavaScript (JS) est un langage de programmation de scripts principalement utilisé côté client pour ajouter de l'interactivité et des fonctionnalités dynamiques aux pages web. Il permet de manipuler le contenu HTML, de réagir aux événements de l'utilisateur , de modifier le style des éléments HTML, de valider les données des formulaires, de créer des animations et bien plus encore.

Javascript

Intermédiaire

PHP (Hypertext Preprocessor) est un langage de script côté serveur utilisé pour développer des applications web dynamiques et interactives. Il est largement utilisé pour générer du contenu HTML dynamique, interagir avec les bases de données, gérer les sessions utilisateur, créer des formulaires web, envoyer et recevoir des cookies, et autre. PHP est souvent intégré dans des fichiers HTML pour permettre la création de pages web dynamiques pour l'utilisateur.

PHP

Amateur

C# ("C sharp") est un langage de programmation orienté objet développé par Microsoft. Il est principalement utilisé pour développer des applications sur la plateforme Microsoft .NET. C# est polyvalent et peut être utilisé pour développer une variété d'applications, . Il offre des fonctionnalités avancées telles que la gestion automatique de la mémoire, la programmation orientée objet, la gestion des exceptions, et le support de la programmation asynchrone. C# est un langage populaire utilisé dans l'industrie du logiciel.

C#

Amateur

Le C++ est un langage de programmation de haut niveau qui offre une combinaison unique de fonctionnalités de programmation orientée objet et procédurale. Il est souvent utilisé pour développer des logiciels nécessitant une grande performance, tels que les jeux vidéo, les logiciels système, les applications graphiques et bien d'autres. Sa syntaxe flexible et sa richesse en fonctionnalités en font un choix populaire parmi les développeurs pour un large éventail d'applications.

C++

Débutant

Photoshop est un logiciel de création et de retouche d'images, largement utilisé par les designers, photographes et professionnels du graphisme. Avec ses outils avancés et ses fonctionnalités de pointe, il permet de manipuler et d'embellir les images numériques pour une multitude d'applications, allant du design graphique à la retouche photo en passant par la création artistique.

Photoshop

Intermédiaire

Adobe Premiere est un logiciel de montage vidéo professionnel largement utilisé par les cinéastes, les vidéastes et les professionnels de l'édition vidéo. Il offre une gamme complète d'outils de montage, d'effets spéciaux, de correction des couleurs, de sonorisation et d'autres fonctionnalités avancées pour créer des productions vidéo de haute qualité.

Ce site est un projet scolaire qui visait à créer un site statique responsive sur une exposition d'Antoni Gaudi. Composé d'un systeme de cookies et de plusieurs rubriques dont d'une biographie, d'une rubrique oeuvre avec 9 images d'ouvrages aléatoire de l'artiste, la lettre commissaire ainsi que des informations sur l'exposition. Le header contient d une video en arrire plan et d un bouton de scroll vers le bas sans compter le titre. La navigation rétrécie passer l'header pour donner plus de place aux informations, elle passe d'une navigation rétrécissante à une navigation en burger à partir de 750 pixels de largeur, un bouton apparaissant après un certain niveau permet de remonter à la tete de la page. La rubrique biographie et composé d'une photo de l'artiste avec à coter une brève biographie, en dessous se trouve plusieurs paragraphes décrivants qui il étaient, son parcours étudiant et professionnel ainsi que sa mort, chaque rubrique peut en afficher plus en détaille avec le bouton plus au-dessus de lui. La rubrique oeuvre contient une vidéo sur l'exposition gaudie avec en dessous 9 oeuvres de Gaudí qui s'affiche aléatoirement sur les 18 disponibles. Ensuite vient la lettre commissaire qui est un explicative de pourquoi l'exposition. Et vient après la section exposition où l'on retrouve diverses informations sur celle-ci comme un carte pour googler map, le different tarif et un lien vers la billetterie. Le footer lui est composé de mes coordonnées personnelles, d'une navigation et la mention légale.

Ce site est conçu en HTML, CSS, Javascript sur une page unique .Le Javascript était surtout utilisé pour les images, et les changement et modification de classe pour les boutons et la navigation.
